# 🤖 Basic Chatbot

A simple rule-based chatbot built using Python that interacts with users through predefined responses. This chatbot is ideal for beginners learning natural language processing, basic Python scripting, or building simple interactive applications.

---

## 📌 Features

- Text-based interaction
- Rule-based responses
- Easy to customize and expand
- Command-line interface
- Lightweight and beginner-friendly

---

## 🛠️ Technologies Used

- Python 3.x
- No external libraries required (pure Python)
- Optional: You can extend it using `NLTK` or `transformers` for advanced capabilities
